كود HTML لتحميل محتوى الوحدة

سطح المكتب

يتم تحميل محتوى Unity في المتصفح بواسطة المكون الإضافي Unity Web Player . عادة لا يتصل كود HTML بهذا البرنامج المساعد مباشرة ولكن من خلال مساعدة برنامج نصي يسمى UnityObject . وتتمثل مهمتها الأساسية في جعل تضمين محتوى Unity مهمة بسيطة جدًا عن طريق حماية المستخدم من المشكلات المختلفة الخاصة بالمتصفح والنظام الأساسي. كما أنه يتيح سهولة تثبيت Web Player.

يحتوي ملف HTML الذي تم إنشاؤه بواسطة Unity عند إنشاء مشغل ويب على جميع الوظائف المطلوبة بشكل شائع. في معظم الحالات، لا يتعين عليك تعديل ملف HTML على الإطلاق. يشرح باقي المستند الأعمال الداخلية لهذا الملف.

يجب تحميل البرنامج النصي UnityObject قبل استخدامه. يتم ذلك في الجزء العلوي من قسم <head> . 

<نوع البرنامج النصي = "نص/جافا سكريبت">
<!--
varUnityObjectUrl = "http://webplayer.unity3d.com/download_webplayer-3.x/3.0/uo/UnityObject.js";
إذا (document.location.protocol == 'https:')
	UnityObjectUrl.replace("http://", "https://ssl-");
document.write('<script type="text/javascript" src="' +UnityObjectUrl + '"></script>');
-->
</script>

يمكن الآن استخدام المتغير UnityObject العام لتنفيذ العديد من المهام المتعلقة بالوحدة. الأهم هو تضمين محتوى الوحدة. يتم تنفيذ ذلك عن طريق استدعاء طريقة embedUnity التي تقبل العديد من المعلمات. يحدد الأول معرف عنصر HTML الذي سيتم استبداله بمحتوى Unity. يمكن أن يكون أي عنصر HTML مع كون <div> هو الأكثر شيوعًا. فكر في الأمر كعنصر نائب مؤقت حيث يجب وضع الوحدة. تحدد المعلمة الثانية المسار إلى ملف بيانات مشغل الويب الذي سيتم عرضه. تحدد المعلمتان التاليتان عرض وارتفاع شاشة العرض لمحتوى مشغل الويب. يمكن أن تكون القيم المقدمة قيم بكسل (على سبيل المثال 600 ، "450" ) أو قيم نسبة مئوية (على سبيل المثال، "50%" ، "100%" ). 

UnityObject.embedUnity("unityPlayer", "WebPlayer.unity3d", 600, 450);

وأخيرًا، يتم وضع العنصر النائب لـ HTML داخل قسم <body> . يمكن أن يكون الأمر بسيطًا مثل <div id="unityPlayer" /> . ومع ذلك، لتحقيق أقصى قدر من التوافق، فمن الأفضل وضع بعض رسائل التحذير في حالة عدم دعم المتصفح لـ JavaScript وعدم استبدال العنصر النائب بـ UnityObject. 

<div معرف = "unityPlayer">
	<div class = "مفقود">
		<a href="http://unity3d.com/webplayer/" title="Unity Web Player. قم بالتثبيت الآن!">
			<img alt="Unity Web Player. قم بالتثبيت الآن!" src = "http://webplayer.unity3d.com/installation/getunity.png" width = "193" height = "63" />
		</أ>
	</div>
</div>
